# bcerq
BitCoin Export ReQuests - explore BTC blockchain SQL DB.

## Requirements

- python3
- python3-psycopg2
- running PostgrSQL server

## Usage

./bcerq.py <options> <command>

Options:
- f --from [date] - starting date
- t --to [date] - end date
- n --num [int] - limit (qty/btc/%)
- a --alist - address list
