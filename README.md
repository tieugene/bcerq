# bcerq
BitCoin Export ReQuests - explore BTC blockchain SQL DB.

## Requirements

- python3
- python3-psycopg2
- running PostgrSQL server

## Usage

./bcerq.py <rq> <options>

Options:
- f --from [date] - starting date
- t --to [date] - end date
- n --num [int] - limit
- o --out output ?
- l --log log to ?
